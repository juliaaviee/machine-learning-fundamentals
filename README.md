# Machine Learning Fundamentals

This repository documents my study journey into **Machine Learning**. It contains practical implementations of fundamental algorithms using **Python** and **Scikit-Learn**, covering both Supervised and Unsupervised learning paradigms.

## Project Structure

The repository is organized by learning topics:

### 1. Supervised Learning (Labeled Data)
* **Classification:** * *Script:* `iris_decision_tree.py`
    * *Algorithm:* Decision Tree Classifier.
    * *Goal:* Predict the species of an Iris flower based on its measurements.
    * *Key Concepts:* Train/Test Split, Accuracy Score, Confusion Matrix.
    
* **Regression:** * *Script:* `housing_linear_regression.py`
    * *Algorithm:* Linear Regression.
    * *Goal:* Predict continuous housing prices using the California Housing dataset.
    * *Key Concepts:* Mean Squared Error (MSE), RÂ² Score, Feature Importance (Coefficients).

### 2. Unsupervised Learning (Unlabeled Data)
* **Clustering:** * *Script:* `iris_clustering.py`
    * *Algorithms:* K-Means, Hierarchical Clustering (Dendrograms).
    * *Goal:* Discover hidden groups/patterns within the Iris dataset without using labels.
    * *Key Concepts:* Elbow Method, Euclidean Distance, Data Normalization (StandardScaler).

## Technologies Used
* **Language:** Python 3.x
* **Libraries:** * `scikit-learn` (Model building)
    * `pandas` & `numpy` (Data manipulation)
    * `matplotlib` & `seaborn` (Data visualization)
